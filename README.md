#Current systems:

# Targeting Pod (TGP)
- Pilot Accessible,full-screen camera that retains flight and weapons control. 
- Information Suite 
  - CRAFT + RETICLE POSITION DATA
  - Target Store + Quick Slew & Reference
- DAY/IR/THERMAL CAPABILITY (with full package w/ dependencies)
- 6 Zoom Steps 
- Slew to Grid/Map
- AREA + POINT GEOLOCK
- HADS Marker Placement + Toggleable overlay. Synced in multiplayer between maps, and other aircrafts overlay.
- IR Sparkle 
- Target Storage & Referencing 
- User is able to store three targets - Slew back to them, and reference them through HMD view.
- Terrain Avoidance System
- Settings configurations for many options.


# Helmet Mounted Display (HMD)
- Information Suite
  - Pitch/Roll Visualizer
  - Speed, Altitude, Heading, Vertical Speed & Fuel
  - TGP Aimpoint Indicator
  - Autohover Indicator
- Weapons Suite
  - Weapons Indicator
  - Dumb Fire Rocket CCIP
- Flight Computer - Visual Indicators + TOF Information
  - RTE System (View Documentation)
    - Automatically develops Flight Route per internal specifications 
  - WP System (Grid OR Map WP)
  - TGT System (From TGP stored target data)
